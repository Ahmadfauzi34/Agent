## 2023-10-27 - [TensorFlow.js Iteration] **Learning:** Repeatedly calling `tf.tidy()` and `dataSync()` inside a JavaScript `.map()` loop (e.g., when calculating cosine similarity for each database entry) is a massive performance bottleneck due to GPU-CPU synchronization overhead. **Action:** Load vector data into a 2D Matrix (`tf.tensor2d`) and use batched linear algebra (e.g., matrix multiplication `matMul`) to compute similarities for the entire database in a single pass.
